键盘快捷键

  1. 加粗（Bold）Ctrl + B

  2. 斜体（Italic）Ctrl + I

  3. 任务列表快速选择 Alt + C

  4. 删除线 Alt + S

  5. 标题（升级）Ctrl + Shift + ]

  6. 标题（降级）Ctrl + Shift + [

  7. 数学环境 Ctrl + M

  8. 预览 Ctrl + Shift + V

  9. 侧边预览 Ctrl + K V

创建目录

  • 在VSCode命令面板中运行命令”Create Table of Contents”来插入一个新目录。(Ctrl + T 输入">Create Table of Contents")
  • 默认情况下，Table of Content（TOC）会自动更新。若要禁用，请更改toc.updateOnSave选项。
  • 轻松添加/更新/删除章节编号
